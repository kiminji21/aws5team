📍 프로젝트 개요
---
다양한 OTT 플랫폼 서비스가 생겨나면서 고객들에게 다양한 OTT 콘텐츠 정보를 한 눈에 볼 수 있고 시청까지 이어지는 프로젝트를 기획했습니다. 이 프로젝트의 목적은 다양한 OTT 정보를 한 플랫폼을 통해 이용할 수 있도록 플랫폼을 구축하고 예상치 못한 과부화로 인해 서비스가 중단되는 것을 방지하는 것이었습니다.
